# Xtream Player

A Flutter streaming platform with Xtream API support.

## Features

- Xtream API integration for movies, series, and live TV
- Profile management with watch progress tracking
- Video player with 4K, Dolby Vision, HDR10+ support
- Multi-platform support (Android, iOS, Linux, macOS, Windows, Web)

## Getting Started

1. Configure your Xtream server in Settings
2. Enter your server URL, username, and password
3. Start streaming!
